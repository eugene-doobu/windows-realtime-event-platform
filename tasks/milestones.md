# Milestones

## M0 - Direction Locked

Done when:

- README reflects the current engine goal
- architecture, validation, roadmap, and backlog are aligned
- public repo boundary is explicit

## M1 - First End-To-End Run

Done when:

- FastAPI can trigger a simulation run
- one subprocess executes a C++ kernel call
- run artifacts are written to disk

## M2 - Large-Scale Population

Done when:

- synthetic population generation supports at least 1000 agents
- one run completes with deterministic seed behavior
- metrics summarize group and channel behavior

## M3 - Benchmark Validation

Done when:

- one synthetic benchmark scenario is included
- three intervention variants can be compared
- validation output clearly passes or fails defined expectations

## M4 - Public Demo Ready

Done when:

- repo contains only domain-neutral assets
- synthetic benchmark can be run from docs
- the repo clearly demonstrates C++ large-scale processing value
